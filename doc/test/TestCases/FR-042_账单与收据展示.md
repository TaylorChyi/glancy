# FR-042 账单与收据展示（Must）

- **前置**
  - 至少存在 3 笔订单（ACTIVE/EXPIRED/REFUNDED），账单接口返回 UTC 时间。
  - 浏览器设置本地时区 UTC+8；抓包工具验证时间转换。
  - PDF/HTML 收据下载地址可访问。
- **步骤**
  1. 进入 UC-15 账单页，验证每条记录显示档位、生效/到期时间、订单号。
  2. 切换界面语言 zh→en，刷新账单，观察文案是否即时翻译。
  3. 点击最近订单的“下载收据”按钮，下载 PDF 与 HTML 并打开校验。
  4. 触发一次账单导出动作，确认日志记录访问行为。
- **期望**
  - 时间戳在 UI 中转换为本地时区显示，后端依旧返回 UTC（可通过响应体验证）。
  - 收据文件包含订单号、金额、税率等信息，编码正确且能追溯；下载操作写入访问日志。
  - 界面语言切换后立即刷新，覆盖率 100%。
- **数据**
  - API：`GET /api/v1/billing/invoices`、`GET /api/v1/billing/invoices/{id}/receipt.pdf`。
  - 事件：`billing.receipt_downloaded`。
  - 订单样例：`INV-202405-001`（ACTIVE）、`INV-202404-002`（EXPIRED）。
- **优先级**：P0（Must）
- **关联 UC**：UC-15
- **关联 AC**：AC-FR-042，AC-UC-15
