# FR-010 释义与义项（Must）

- **前置**
  - Plus 用户登录并选择语言对 `en→zh`。
  - 准备高多义词条 `set` 与低义项词条 `thermodynamics`。
  - 在 DevTools Mock `modules.examples` 请求失败以验证隔离性。
- **步骤**
  1. 查询 `set`，记录默认展示的义项数量与字符长度。
  2. 点击“展开更多”，继续滚动确认排序是否保持一致。
  3. 切换语言对为 `zh→en`，查询 `热力学`，观察字符上限是否符合英文 220 的约束。
  4. 模拟例句模块 500，确认释义模块仍然渲染且给出“例句暂不可用”提示。
- **期望**
  - 默认呈现至少 3 个义项，详略级别为“中”，中文释义 ≤150 字，英文释义 ≤220 字符。
  - 展开后新增义项附加在尾部，原有顺序不发生重排。
  - 跨语言对查询时字段齐全（`definition`, `explanation`, `posTag` 等），并根据语言自动切换字符限制。
  - 其他模块失败时释义模块不受影响，页面展示模块级错误提示。
- **数据**
  - API：`POST /api/v1/lookup`，`modules` 字段包含 `definitions`。
  - 词条：`set`（多义），`热力学`。
  - 验证字段：`modules.definitions[*].displayLevel`、`modules.definitions[*].textLength`。
- **优先级**：P0（Must）
- **关联 UC**：UC-01，UC-12
- **关联 AC**：AC-FR-010，AC-UC-01，AC-UC-12
