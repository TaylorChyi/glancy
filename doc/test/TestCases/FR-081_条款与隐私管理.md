# FR-081 条款与隐私管理（Must）

- **前置**
  - 新用户账号 1 个；已有账号 1 个（可直接访问设置）。
  - 隐私条款版本号 `consentVersion` 可供校验。
  - 数据导出/删除指引链接在设置页配置。
- **步骤**
  1. 新用户首次登录时不勾选条款/隐私，点击“继续”，验证被阻止；随后勾选，完成登录，并记录 `consentVersion`。
  2. 在设置页找到“撤回同意”入口，点击后确认系统提示导出与删除指引链接且自动切换到无痕模式。
  3. 再次进入设置页，查看 `consentStatus`，应显示“已撤回”并提供重新同意入口。
  4. 触发数据导出或删除流程，确认日志中包含 `consent` 状态，满足可审计性。
- **期望**
  - 未勾选条款/隐私时无法继续；勾选后在审计日志中写入 `consent.accepted`，含版本号与时间。
  - 撤回同意会立即禁用历史留存，并展示导出/删除指引；日志记录 `consent.revoked`。
  - 状态在后续会话中保持，并可重新同意；审计系统可查看所有状态变更。
- **数据**
  - API：`POST /api/v1/consent`、`DELETE /api/v1/consent`。
  - 事件：`consent.accepted`, `consent.revoked`（含 `version`、`actor`）。
  - 链接：导出说明、删除流程 Runbook。
- **优先级**：P0（Must）
- **关联 UC**：UC-10，UC-16
- **关联 AC**：AC-FR-081，AC-UC-10，AC-UC-16
