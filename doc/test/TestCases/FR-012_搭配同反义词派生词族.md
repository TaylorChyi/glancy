# FR-012 搭配、同反义词、派生词族（Must）

- **前置**
  - Pro 用户登录，语言对 `en→zh`。
  - 准备高频词 `run` 与低频词 `ululation` 以覆盖空数据场景。
  - Network 面板开启 to capture `modules.phrases`, `modules.synonyms`, `modules.derivatives`。
- **步骤**
  1. 查询 `run`，记录常见/固定搭配、同义词、反义词、派生词族的默认条数。
  2. 点击“更多搭配”“更多同义词”等按钮，确认分页加载方式及排序稳定性。
  3. 查询 `ululation`，观察模块是否以空集合呈现且无错误弹窗。
- **期望**
  - 默认展示满足最低条数：常见搭配/固定搭配各 ≥3，同/反义词各 ≥4，派生词族 ≥2；结果按相关度排序且去重。
  - “更多”只向后追加新条目，不重排已展示内容，且分页请求参数可追踪。
  - 缺少数据时模块以空状态显示，schema 返回空数组，无 fallback 错误。
- **数据**
  - API 字段：`modules.phrases`、`modules.collocations.fixed`、`modules.synonyms`、`modules.derivatives`。
  - 观测事件：`module.load_more` 包含 `moduleId` 与 `page`。
  - 词条：`run`、`ululation`。
- **优先级**：P0（Must）
- **关联 UC**：UC-01，UC-03，UC-12
- **关联 AC**：AC-FR-012，AC-UC-01，AC-UC-03，AC-UC-12
