# FR-060 主题、快捷键与界面语言（Must）

- **前置**
  - 浏览器支持 prefers-color-scheme；账号登录 UC-07 设置页。
  - 准备两个设备（桌面/移动）验证同步。
  - 记录默认快捷键表，确保可测试冲突提示。
- **步骤**
  1. 在桌面设备切换主题为“跟随系统”，随后在操作系统层切换亮/暗模式，观察页面是否无闪烁地随动。
  2. 新增自定义快捷键 `Ctrl+K` 绑定“聚焦输入框”，再尝试设置 `Ctrl+C`（应冲突）查看提示。
  3. 切换界面语言 zh→en，检查 UI 文案是否即时刷新，并在移动端同步该语言设置。
- **期望**
  - 主题切换不出现白屏/闪烁，`prefers-color-scheme` 生效，跨端同步后移动端也保持同样模式。
  - 快捷键保存前进行冲突检测，禁止覆盖系统组合键并显示冲突说明；合法快捷键保存后立即可用。
  - 语言切换后 UI 文案 100% 更新，无遗留中文/英文；设置同步至其它设备。
- **数据**
  - API：`PATCH /api/v1/settings/ui`（字段 `theme`, `shortcuts`, `locale`）。
  - 事件：`settings.ui.updated`，`shortcut.conflict`。
  - 验证工具：PA11y / Storybook 可用于检测主题切换闪烁。
- **优先级**：P0（Must）
- **关联 UC**：UC-07
- **关联 AC**：AC-FR-060，AC-UC-07
