## 全局生成规则
1) 输出顺序、标题层级与标点必须与“输出模板”完全一致，禁止新增、重排或省略模板中要求保留的章节。缺少可靠内容的章节直接整体删除，不保留标题。
2) 必须至少输出 `# 单词：{word}`、`## 音标`，需包含英式与美式两行，以及 `## 释义与例句` 中的 `s1`，且 `s1` 至少包含 `min_examples_per_sense` 条例句。
3) 音标仅使用 IPA，即 International Phonetic Alphabet 国际音标，用斜杠包裹，例如 /ˈrʌn/，并标出主重音。
4) 义项编号固定为 `s1, s2, s3, ...`，并使用模板中的标题格式：`### s# 释义：{part_of_speech_abbr}. {part_of_speech_full} — {中文解释}`。
5) 例句遵循统一结构：
   - **例句 1**: {英文句子}
     **翻译**: {中文翻译}
   - 每个义项的例句数量介于 `min_examples_per_sense` 与 `max_examples_per_sense`；例句需原创、自然且互不重复，章节之间不得复用。
6) 将“常见搭配 / 常见词组 / 固定习语 / 固定句型 / 近义词 / 反义词 / 专业领域用法 / 易混淆词”中可明确绑定到义项的条目，在条目末尾标注 `[对应：s#]`；无法绑定时省略该标注。
7) 词汇学信息中的语体标签仅从 `{正式, 中性, 非正式, 口语, 书面, 俚语, 方言, 学术}` 中多选；地域标签仅从 `{英式, 美式, 兼用, 其他}` 中选择其一。不确定时整项删除。
8) CEFR，即 Common European Framework of Reference for Languages 欧洲语言共同参考框架，与词频标签仅在信息可靠时出现；不可靠则连同行删除。
9) 所有缩写在首次出现时必须附带全拼，例如：IPA 即 International Phonetic Alphabet 国际音标、CEFR 即 Common European Framework of Reference for Languages 欧洲语言共同参考框架、POS 即 Part of Speech 词性。
10) 禁止输出表格、代码块、额外前后缀文本、引用块、脚注或自动生成的总结。
11) 最后一行必须独占输出哨兵 `<END>`，且其前一行必须为正文内容，不得留有多余空格。
12) 英文句子必须保持自然的单词间空格，禁止将单词或标点紧贴合并。
